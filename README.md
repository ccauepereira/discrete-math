# 🧠 Discrete Math Java

Projeto acadêmico focado na aplicação prática de Matemática Discreta utilizando Java.

O objetivo deste repositório é transformar conceitos matemáticos em implementações computacionais simples, didáticas e organizadas.

O projeto utiliza:

* Java puro
* lógica matemática
* estruturas computacionais básicas
* Scanner
* vetores
* loops
* funções matemáticas
* implementação direta de fórmulas
* resolução de questões universitárias

---

# 📌 Padrão dos Códigos

Todos os códigos seguem o mesmo padrão:

* tudo minúsculo
* sem package
* sem Main.java
* uma classe pública por arquivo
* nome da classe igual ao nome do arquivo
* IntelliJ usando src como Source Root
* sem arquitetura enterprise
* sem abstrações exageradas
* foco didático e matemático

# 🚀 Como executar

Clone o repositório:

```bash id="m5e8z2"
git clone https://github.com/ccauepereira/discrete-math
```

Entre na pasta:

```bash id="b4r7x1"
cd discrete-math
```

Compile um arquivo:

```bash id="u3f6k9"
javac src/inducao/somanaturais.java
```

Execute:

```bash id="n2p4c8"
java -cp src/inducao somanaturais
```
